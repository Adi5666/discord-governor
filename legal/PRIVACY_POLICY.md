# Privacy Policy

**Last Updated:** [28/01/2026]

This Privacy Policy explains how **Discord Governor** (“the Service”, “the Bot”, “we”, “us”) collects, uses, and protects information.

By using the Service, you agree to this policy.

---

## 1. Information We Collect

We only collect data that is necessary for the Service to function.

### a) Automatically Collected
- Discord User IDs
- Discord Server (Guild) IDs
- Role and permission identifiers
- Subscription status
- Configuration settings
- Audit log metadata (actions, timestamps)

### b) Technical Data
- IP address (for security & abuse prevention)
- User agent (dashboard sessions only)

---

## 2. What We Do NOT Collect

We do **not**:
- Read or store private Discord messages
- Access message content outside command usage
- Collect passwords
- Sell or rent user data
- Track users across other services

---

## 3. How We Use Data

Data is used only to:
- Provide bot functionality
- Enforce permissions & subscriptions
- Maintain audit logs
- Prevent abuse and fraud
- Provide dashboard access
- Improve service stability

---

## 4. Data Storage & Security

- Data is stored securely in a managed database
- Access is restricted to authorized systems
- Industry-standard security practices are used
- No system is 100% secure, but we minimize risk

---

## 5. Data Retention

- Server data is retained while the bot is in the server
- Audit logs may be retained for security and compliance
- Data may be deleted upon bot removal or account termination

---

## 6. Third-Party Services

We may use trusted third-party services for:
- Hosting (e.g., Render)
- Databases
- Payment processing (future)

These services are only given access necessary to operate.

---

## 7. User Rights

You may:
- Request data deletion
- Remove the bot from your server
- Cancel subscriptions

Some data may be retained for legal or security reasons.

---

## 8. Children’s Privacy

The Service is not intended for users under **13 years of age**.  
We do not knowingly collect data from children.
